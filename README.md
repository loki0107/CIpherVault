## Overview
CipherVault is a modern, dark-mode GUI application for secure text encryption. It uses the **AES-128 (Fernet)** algorithm to transform readable text into ciphertext, ensuring secure communication and data storage.

## Features
- **Military-Grade Encryption:** Uses symmetric AES-128 via the `cryptography` library.
- **Modern UI:** Built with CustomTkinter for a sleek, dark-themed experience.
- **Key Management:** Auto-generates and handles secure keys.

## Usage
1. Run the app: `python crypto_app.py`
2. Click **Generate Key**.
3. Type your message and click **Encrypt**.
